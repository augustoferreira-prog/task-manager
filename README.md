# 📝 TaskFlow

Aplicación web para gestionar tareas de forma sencilla y rápida, con persistencia de datos en el navegador.

---

## 🚀 Demo

https://tu-proyecto.vercel.app

---

## ✨ Funcionalidades

* Añadir tareas
* Eliminar tareas
* Marcar como completadas
* Filtrar tareas (todas, completadas, pendientes)
* Buscar tareas
* Categorías y prioridades
* Estadísticas y barra de progreso
* Modo oscuro
* Persistencia con LocalStorage

---

## 🛠️ Tecnologías

* HTML
* Tailwind CSS
* JavaScript
* LocalStorage

---

## 📦 Instalación

```bash
git clone https://github.com/tuusuario/taskflow.git
cd taskflow
```

Abrir `index.html` en el navegador.
